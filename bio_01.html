1.1 腎元（Nephron）功能與各段通透性 ★★★ 必考
部位水的通透Na⁺/溶質調控因子機制
腎小球（過濾）通過全部通過（小分子）腎絲球靜水壓超過濾
近端小管 PCT✅ 被動回收（osmosis）Na⁺ 主動回收（65%）—主動轉運後水跟隨
亨耳氏下降細端✅ 被動流出溶質不通透（Na⁺留著）—髓質高滲 → 水自由擴散出
亨耳氏上升細端❌ 不通透Na⁺ 被動流出—濃度梯度驅動 NaCl 出
亨耳氏上升粗端❌ 絕對不通水Na⁺ 主動泵出（NKCC2）Loop 利尿劑標靶建立髓質濃度梯度的關鍵
遠端小管 DCT有限（受ADH）NaCl 主動回收AldosteroneThiazide 利尿劑標靶
集尿管✅ 被動（受ADH ↑）Na⁺ 主動回收ADH→Aquaporin-2ADH↑→插入 AQP2→水回收↑
粗上升端（thick ascending limb）對水完全不通透，Na⁺主動泵出才能建立髓質高滲梯度。選項說「粗上升端水可被動回收」→ 錯誤！（中國醫 106 第 45 題 C）
ADH（抗利尿激素）作用機制 高滲透壓/低血量 → 下視丘感應 → ADH 由腦垂腺後葉釋放 → 集尿管 V2 受體（GPCR）→ cAMP↑ → PKA → Aquaporin-2 插入頂端膜 → 水回收↑ → 尿量↓
Aldosterone 作用機制 低血量/低Na⁺ → RAAS→ Aldosterone（腎上腺皮質）→ 遠端小管&集尿管 → 促進 ENaC（Na通道）與 Na/K-ATPase 表現 → Na⁺ 回收↑、K⁺ 排出↑、H⁺ 排出↑
HCO₃⁻ 再吸收（碳酸氫根） 大部分（~80-90%）在近端小管再吸收（義守 104 第 34 題）。腎小管細胞中 CO₂ + H₂O → H₂CO₃ → H⁺ + HCO₃⁻，H⁺ 分泌至管腔，HCO₃⁻ 回到血液。
1.2 心臟生理 ★★★ 必考
心臟傳導系統（順序）
竇房結 SA node → 房室結 AV node → His 束 → Purkinje 纖維 → 心室肌
心肌鈣離子觸發機制（CICR） 動作電位 → L-type Ca²⁺ channel 開（胞外Ca²⁺進入）→ 觸發肌漿網（SR）大量釋放 Ca²⁺（RyR receptor）→ troponin C 結合 Ca²⁺→ actin-myosin 結合 → 收縮
特性骨骼肌心肌
自律性無✅ 有
完全強直收縮✅ 可以❌ 不能
不應期 vs 收縮期不應期短不應期 ≥ 收縮期
細胞連接無 gap junctionintercalated disc（gap junction）

心肌不會完全強直收縮（tetanus）。心肌的不應期特別長（覆蓋整個收縮期），因此動作電位無法在收縮結束前再次觸發，這是防止心臟「鎖死」的保護機制。（中國醫 106 第 39 題 D 為錯誤選項）

迷走神經對心臟的作用 迷走神經（副交感）→ 釋放 ACh（乙醯膽鹼）→ 心肌 M2 受體（GPCR）→ Gᵢ蛋白 → cAMP↓ + K⁺通道打開 → 心跳率↓（負性變時效應）。（中國醫 106 第 39 題 A）

1.4 神經系統生理 ★★
動作電位（Action Potential）機制 靜止膜電位 ≈ −70mV（K⁺通透，Na⁺主動泵出）
去極化：Na⁺快速內流（電壓門控 Na⁺通道開）
再極化：K⁺外流（電壓門控 K⁺通道開、Na⁺通道關）
後過極化（hyperpolarization）：K⁺通道延遲關閉
不應期：Na⁺通道處於失活狀態
突觸後電位加成 空間加成（spatial summation）：多個突觸同時輸入
時間加成（temporal summation）：同一突觸連續輸入（高醫 105 慈濟 105）
IPSP（抑制性）：K⁺或Cl⁻通透性增加 → 膜電位更負 → 難以觸發 AP（義守 104 第 18 題）
超極化（Hyperpolarization）特徵 膜電位比靜止電位更負（如 −80mV）
K⁺ 和/或 Cl⁻ 通道開放
此時更難觸發動作電位
高醫 105 第 55 題：超極化時 Na⁺/Cl⁻ 通道打開→此說法錯誤（正確為 K⁺ 或 Cl⁻）
自主神經系統 副交感（迷走）：ACh → M受體 → 心跳↓、消化↑、支氣管縮小、唾液↑
交感：NE/Epi → α/β受體 → 心跳↑、消化↓、支氣管擴大
刺激副交感 → 唾液分泌↑（慈濟 105 第 34 題）
感官生理要點 視覺：桿細胞（rod）暗光；錐細胞（cone）色覺。Rhodopsin = G-protein coupled receptor（GPCR）。光照 → rhodopsin 活化 → transducin（Gi）→ cGMP↓ → 陽離子通道關 → 超極化 → 麩胺酸釋放↓（高醫 105 第 18 題）。
盲點：神經節細胞軸突聚集處，無感光受器（義守 104 第 20 題）
Umami 受體：偵測麩胺酸鈉（MSG）（義守 104 第 21 題）

主動脈（動脈血）：PO₂ ≈ 100 mmHg；PCO₂ ≈ 40 mmHg

2.2 RNA 轉錄後加工（Post-transcriptional Processing）★★★ 必考
5' Capping（加帽） 5'端以 5'-5' 磷酸二酯鍵加上 7-methylguanosine（m7G）帽
功能：防止 5' exonuclease 降解、協助核糖體辨識、促進核輸出
Splicing 三種機制對比
類型起始方式特色
Group I Intron外加 G（exogenous G）起始第一個酯交換Self-splicing ribozyme
Group II Intron內在 A（branch point A）起始Self-splicing；與 spliceosome 機制類似
Spliceosome內在 A（branch point A）起始snRNA（U1/U2/U4/U5/U6）形成複合體
Group I 用外加 G；Group II 和 Spliceosome 均用內在 A。中國醫 106 第 29 題（D）說「spliceosome 有 RNA 5'-5' 連結」→ 錯誤（只有 5' capping 才是 5'-5' 連結）
Intron 固定特徵（GT-AG rule） 5' 端：GU（DNA 層面：GT）
3' 端：AG
Branch point A：位在 3' 端上游 20-50 nt 處
中國醫 106 第 35 題：5'端第一個 G、第二個 U（選項 A 正確）
Polyadenylation（加多聚腺苷酸） 辨識序列：AAUAAA（在 poly-A 訊號下游切斷）→ 加上 ~200 個 A
功能：保護 3' 端不被降解、促進核輸出、協助翻譯
Ribozyme（核酶）催化的反應 RNA 本身具有酵素活性：
① Group I/II intron splicing（自我剪切）
② 核糖體 rRNA（23S）催化胜肽鍵形成（中國醫 106 第 34 題 E 正確）
③ RNase P 切割 pre-tRNA
2.3 DNA 突變類型與 DNA 修復 ★★★ 必考
突變類型定義影響
Silent mutation密碼子改變但胺基酸不變無影響（遺傳密碼子退化性）
Missense mutation密碼子改變 → 不同胺基酸蛋白質功能可能改變
Nonsense mutation密碼子 → 終止密碼子（UAA/UAG/UGA）蛋白質提早終止
Frameshift mutation插入/缺失 → 讀碼框改變後續多個胺基酸全部改變
Base transversion嘌呤↔嘧啶互換點突變
Base transition嘌呤↔嘌呤 或 嘧啶↔嘧啶點突變（較常見）
Missense mutation 不會導致過早終止！Nonsense mutation 才會。（中國醫 106 第 11 題 B 為錯誤選項）
DNA 修復機制修復傷害相關疾病
BER（Base excision repair）氧化鹼基、去胺基、甲基化—
NER（Nucleotide excision repair）嘧啶二聚體（UV）、大型加成物著色性乾皮症（XP）
MMR（Mismatch repair）複製錯配、插入/缺失Lynch syndrome（大腸癌）
DSBR/HR（同源重組）雙股斷裂BRCA1/2 乳癌
NHEJ（非同源末端接合）雙股斷裂（錯誤率較高）免疫缺陷
BRCA1 → DSBR（雙股斷裂修復） 中國醫 106 第 41 題：BRCA1 主要參與雙股斷裂修復（DSBR）（答案A），不是 NER 或 MMR。

6.1 Hardy-Weinberg 平衡計算
p² + 2pq + q² = 1 (p+q=1)
p: 顯性頻率 | q: 隱性頻率 | 2pq: 攜帶者頻率
例題：鐮刀血球 (中醫 106) 發生率 144/1,000,000 → q² = 0.000144 → q = 0.012 → p = 0.988
