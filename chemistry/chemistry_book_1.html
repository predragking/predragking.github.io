立體化學 Stereochemistry

R/S Configuration · E/Z Isomerism · Projection Formulas · Special Compounds
~12%

1.1 CIP 優先序規則與 R/S 標記 CIP PRIORITY RULES

📋 R/S 標記四步驟（每年必考）
  1. 原子序排優先序：直接連接手性碳的原子，原子序大者優先（I > Br > Cl > S > F > O > N > C > H；孤對電子排最低）。
  2. 同原子序時，比較下一層：往外展開（多個分支時取原子序最大者），直到分出高低。雙鍵視為兩個虛擬單鍵（如 C=O 視為 C 連兩個 O）。
  3. 最低優先基移至遠端：H 或最低優先基需指向遠離觀察者（虛線楔形）。
  4. 判斷 1→2→3 的旋轉方向：順時針 → R；逆時針 → S。
⚠ 最高頻陷阱：最低優先基在楔形（朝向觀察者）
若 H（或最低優先基）在楔形鍵（面向觀察者）而非虛線鍵，
觀察到的旋轉方向與真實方向相反，需要翻轉結果：
觀察順時針 → 實際為 S；觀察逆時針 → 實際為 R。

1.2 E/Z 幾何異構 E/Z ISOMERISM

Z — zusammen（在一起）
雙鍵兩端，各自的高優先基在同側
記憶：Z = 相 Zusammen（相聚）
類似 cis，但以 CIP 規則判斷
E — entgegen（對面）
雙鍵兩端，各自的高優先基在異側
記憶：E = 敵 Entgegen（對立）
類似 trans，但以 CIP 規則判斷

1.3 投影式的使用規則 PROJECTION FORMULAS

投影式水平鍵方向垂直鍵方向操作規則主要用途
Fischer 投影朝向觀察者（楔形）遠離觀察者（虛線）旋轉 180° 構型不變；旋轉 90° 構型改變胺基酸、糖類
Newman 投影前碳（圓點中心）後碳（圓圈邊緣）前碳與後碳的立體關係構象分析（交叉/重疊）
Wedge-Dash楔形（實線）= 朝前虛線 = 朝後最直觀三維表示通用立體化學
💡 Fischer 投影快速判斷構型的技巧
Fischer 投影中，任意交換兩個取代基偶數次 → 構型不變；奇數次 → R/S 翻轉。
常用方式：若需判斷，可將 H 換到最下方（垂直）→ 此時水平 3 個基的排列即為真實 R/S 旋轉方向。

1.4 立體異構體類型辨析 STEREOISOMER CLASSIFICATION

名稱定義光學活性物理性質典型例子
對映體 Enantiomers彼此互為不可重疊鏡像有（方向相反）除旋光方向外完全相同(R)- vs (S)-乳酸
非對映體 Diastereomers立體異構但非鏡像關係各自有（不同程度）物理性質各異（bp, mp 均不同）erythro vs threo 型
內消旋物 meso有手性中心，但整體有內部對稱面無（光學非活性）不同於外消旋混合物（純化合物）meso-酒石酸
外消旋混合物 Racemate等量 (R)+(S) 的混合物無（互相抵消）熔點通常與純對映體不同SN1 產物（碳陽離子）
⚠ meso 化合物 ≠ 外消旋混合物
meso 化合物是單一純化合物，具有內部對稱面（手性中心互相抵消），光學非活性但不可被光學拆分。
外消旋混合物是 R:S = 1:1 的混合物，可被手性試劑拆分為兩個對映體。

1.5 質子的立體化學環境 PROCHIRALITY

Homotopic（等位）
  • 取代任一個 → 得到同一化合物
  • 任何環境中 NMR 均等價（相同 δ）
  • 例：CH₄ 的 4 個 H；CH₂Cl₂ 中的兩個 H（⚠ 此為 enantiotopic！）
Enantiotopic（對映位）
  • 取代其中一個 → 得到對映體
  • 非手性環境 NMR 等價；手性環境不等價
  • 例：CH₂Cl₂ 的兩個 H（取代後 → R/S）
Diastereotopic（非對映位）
  • 取代任一個 → 得到非對映體
  • 任何環境中 NMR 均不等價（δ 不同）
  • 例：手性碳旁的 CH₂ 兩個 H（最常考！）

2.1 SN1 vs SN2 完整比較 NUCLEOPHILIC SUBSTITUTION

比較項目SN1（單分子）SN2（雙分子）
速率定律Rate = k[RX] （一級）Rate = k[RX][Nu] （二級）
底物偏好3° > 2° ≫ 1°（碳陽離子穩定性決定）1° > 2° ≫ 3°（空間障礙決定）
中間體碳陽離子（planar sp²，可重排）無中間體（同步協調機制）
立體化學外消旋化（Racemization）：碳陽離子兩面均可攻擊構型翻轉（Walden Inversion）：背面攻擊
溶劑偏好極性質子溶劑（H₂O, ROH, RCOOH）穩定離子極性非質子溶劑（DMSO, DMF, CH₃CN）增強親核力
親核劑弱親核劑亦可（甚至 H₂O）需強親核劑（CN⁻, I⁻, N₃⁻, RS⁻）
副反應消去（E1）競爭；碳陽離子重排不重排；可能有 E2 競爭（強鹼性親核劑）
溫度效應升溫利於 SN1/E1升溫利於消去（E2 比 SN2 優勢增大）
── 離去基能力（Leaving Group Ability）──
TsO⁻ ≫ I⁻ > Br⁻ > Cl⁻ ≫ F⁻
── 親核力（在質子溶劑中，極化性 > 電負度）──
I⁻ > Br⁻ > Cl⁻ > F⁻ ； HS⁻ > HO⁻ ； RS⁻ > RO⁻
── 注意：OH⁻, OR⁻, NH₂⁻ 為不良離去基，需質子化（H⁺）或活化後才能離去 ──

2.2 反應路徑決策流程 DECISION FLOW

▶ 判斷底物 RX 的級數，再看試劑性質

底物為 1° RX？
強 Nu + 非質子溶劑 → SN2
強鹼（t-BuOK, LDA） → E2（Hofmann 產物）
底物為 2° RX？
強 Nu + 非質子溶劑 + 低溫 → SN2（主）
強鹼 + 高溫 → E2（主）
弱 Nu + 質子溶劑 → SN1 / E1 競爭
底物為 3° RX？
強鹼 + 高溫 → E2（Zaitsev 或 Hofmann）
質子溶劑 + 熱 → SN1 / E1
強 Nu 攻擊 3° 碳：空間阻礙太大 → SN2 不可能

2.3 E1 vs E2 消去反應 ELIMINATION REACTIONS

比較項目E1E2
速率定律Rate = k[RX]（一級）Rate = k[RX][base]（二級）
鹼的要求弱鹼或溶劑即可需強鹼（NaOEt, t-BuOK, NaH）
立體化學要求無特定要求（碳陽離子中間體）anti-periplanar H 與 LG（反式共面必需）
區域選擇性通常 Zaitsev（多取代烯）小鹼 → Zaitsev；大位阻鹼 → Hofmann
溫度適中溫度高溫利於 E2（vs SN2）
🔑 E2 的 anti-periplanar 要求——在環己烷上的關鍵應用

E2 消去需要 H 和離去基（LG）處於 anti-periplanar（反式共面） 位置，即二面角 ≈ 180°。

環己烷構象的影響：
  • LG 在 axial（軸向）：相鄰碳的 axial H 與其 anti-periplanar → E2 快速
  • LG 在 equatorial（赤道）：找不到 anti-periplanar H → E2 極慢甚至不發生
  • 因此：trans-2-溴-1-甲基環己烷的 E2 速率 >> cis 異構體
Zaitsev vs Hofmann：
  • Zaitsev：小鹼（NaOEt）→ 多取代烯（熱力學穩定）
  • Hofmann：大位阻鹼（t-BuOK, LDA）→ 少取代烯（動力學）
  • 高溫有利於消去；低溫、強親核劑有利於取代
SN/E 選擇速記 強鹼 + 高溫 = E2 ｜ 弱Nu + 質子溶劑 = SN1/E1 ｜ 強Nu + 非質子 + 1°/2° = SN2 ｜ 3°底物強鹼 = E2
練習題 — SN/E 路徑判斷★★★ 超高頻

下列哪個反應主要進行 SN2？

A. (CH₃)₃CBr + H₂O（25°C）
B. CH₃CH₂CH₂Br + NaCN（DMSO，25°C）
C. (CH₃)₂CHBr + t-BuOK（EtOH，80°C）
D. (CH₃)₃CBr + NaI（丙酮）

✅ 答案：B
A → 3° 底物 + 水 → SN1 ／ E1
B → 1° 底物 + 強 Nu（CN⁻）+ 非質子溶劑 DMSO → SN2 ✓
C → 2° 底物 + 大位阻強鹼 + 高溫 → E2（Hofmann 產物）
D → 3° 底物，即使強 Nu，SN2 空間阻礙過大 → E2 或 SN1

🔑 Markovnikov 規則的本質原因（不是死背！）

表面定義：H 加到含 H 較多的碳，X 加到含 H 較少的碳（Markovnikov 1870）

本質理由：H⁺ 加成後形成碳陽離子，穩定性決定優勢路徑：
3° 碳陽離子 > 2° > 1° > CH₃⁺（超共軛穩定）
因此 H⁺ 加到「少取代端」→ 碳陽離子在「多取代端」→ 更穩定 → 親核劑在此攻擊

⚠ 特殊情況：烯丙位/苄位碳陽離子
若形成的碳陽離子可共振（烯丙位、苄位），穩定性更高，可能改變加成方向或產生重排。

3.2 芳香族親電取代 (EAS) ELECTROPHILIC AROMATIC SUBSTITUTION

反應名稱試劑真正的親電試劑重要條件
硝化 NitrationHNO₃ + H₂SO₄（混酸）NO₂⁺（硝基陽離子）H₂SO₄ 質子化 HNO₃ → NO₂⁺ + H₂O（⚠ 常考！）
磺化 Sulfonation發煙 H₂SO₄（SO₃/H₂SO₄）SO₃ 或 HSO₃⁺可逆反應（高溫 → 脫磺酸）
鹵化 HalogenationX₂ + Lewis acid（FeBr₃, AlCl₃）X⁺（活化的鹵素）FeBr₃ + Br₂ → [FeBr₄]⁻ + Br⁺
FC 烷基化RCl + AlCl₃R⁺（或碳陽離子複合物）⚠ 可能碳陽離子重排；多烷基化問題
FC 醯基化RCOCl + AlCl₃RCO⁺（醯基陽離子）不重排（共振穩定）；得到酮再 Clemmensen 還原
⚠ 高頻考試陷阱：硝化反應的親電試劑不是 HNO₃！
反應機制：HNO₃ + H₂SO₄ → NO₂⁺ + HSO₄⁻ + H₂O
真正攻擊苯環的親電試劑是 NO₂⁺（硝基陽離子），HNO₃ 只是前驅物。
義守 114 Q39 即考此題，答案為 NO₂⁺。

3.3 取代基定位效應 DIRECTING EFFECTS

鄰/對位導向基（EDG，活化苯環）
強活化：-NH₂, -NHR, -NR₂, -OH, -OR
弱活化：-R（烷基）, -Ph（苯基）
特殊：鹵素（-F, -Cl, -Br, -I）→ 鄰/對位導向但鈍化（誘導 vs 共振）

原因：EDG 透過共振在鄰/對位穩定帶正電的碳陽離子中間體（Wheland 中間體）
間位導向基（EWG，鈍化苯環）
強鈍化：-NO₂, -NR₃⁺, -CF₃
中等鈍化：-CN, -CHO, -COR, -COOH, -COOR
弱鈍化：-SO₃H

原因：EWG 在鄰/對位去穩定碳陽離子中間體，間位受影響較小

3.4 Hückel 規則與芳香性 HÜCKEL RULE

芳香性判斷四條件（全部符合才算芳香）
  1. 環狀（cyclic）
  2. 平面（planar，sp² 或 sp）
  3. 完全共軛（fully conjugated）
  4. π 電子數 = 4n+2（n=0,1,2…；即 2, 6, 10, 14…個）
重要例子：
  • 苯（6π, n=1）✅
  • 吡啶 Pyridine（6π, N孤對在sp²）✅
  • 吡咯 Pyrrole（6π, N孤對進入共軛）✅
  • 環戊二烯陰離子 C₅H₅⁻（6π）✅
  • 環辛四烯（8π, 4n 反芳香）❌

5.1 Gibbs 自由能與自發性 GIBBS FREE ENERGY

── 核心三角關係（三式任知其一可推算其餘）──
ΔG° = −RT ln Keq （熱力學 ↔ 平衡）
ΔG° = −nFE°cell （熱力學 ↔ 電化學）
ΔG = ΔH − TΔS （標準或非標準）
ΔG = ΔG° + RT ln Q （非標準條件；Q = 當前反應商）

── 自發性判斷 ──
ΔG < 0 → 正向自發 ΔG = 0 → 平衡 ΔG > 0 → 非自發（逆向自發）
ΔHΔSΔG = ΔH − TΔS自發性條件
−（放熱）+（熵增）永遠 < 0任何溫度均自發 ✅
+（吸熱）−（熵減）永遠 > 0任何溫度均非自發 ❌
−（放熱）−（熵減）低溫 < 0 / 高溫 > 0低溫自發（|ΔH| > T|ΔS|）
+（吸熱）+（熵增）低溫 > 0 / 高溫 < 0高溫自發（T|ΔS| > |ΔH|）
⚠ ΔG 與 ΔG° 的區別（高頻易錯）
ΔG°（標準自由能）：標準狀態（25°C, 1 atm, 1 M），與 Keq 連結：ΔG° = −RT ln Keq
ΔG（任意狀態）：ΔG = ΔG° + RT ln Q，當 ΔG = 0 時系統達平衡（非 ΔG° = 0）
題目問「與平衡常數的關係」→ 討論的是 ΔG°，不是 ΔG

5.2 Hess 定律焓計算 HESS'S LAW

三種計算方式
方程式加減法
將已知反應倒轉（ΔH 變號）或乘以係數（ΔH 等比例），相加得目標方程式
標準生成焓法（★★★）
$\Delta H_{rxn} = \sum \Delta H_f°(\text{products}) - \sum \Delta H_f°(\text{reactants})$
元素穩定態的 ΔHf° = 0
鍵能估算法（★★）
$\Delta H \approx \Sigma BE(\text{斷鍵, +}) - \Sigma BE(\text{成鍵, -})$
斷鍵吸熱（+）；成鍵放熱（−）

5.3 Kc 與 Kp 的換算 Kc vs Kp

Kp = Kc (RT)^Δn
Δn = 氣態產物莫耳數 − 氣態反應物莫耳數 （固態、液態不計入！）

例1：N₂(g) + 3H₂(g) ⇌ 2NH₃(g) → Δn = 2 − (1+3) = −2
Kp = Kc(RT)⁻² = Kc / (RT)²

例2：CO(g) + ½O₂(g) ⇌ CO₂(g) → Δn = 1 − (1+0.5) = −0.5
Kp = Kc(RT)^(−0.5)

當 Δn = 0 時，Kp = Kc（如 H₂ + I₂ ⇌ 2HI）

5.4 Le Chatelier 原則完整整理 LE CHATELIER'S PRINCIPLE

操作平衡移動方向Keq 是否改變補充說明
增加反應物濃度向正反應（右）❌ 不變—
移除產物向正反應（右）❌ 不變—
增加壓力（減少體積）向氣態 mol 數較少的一端❌ 不變Δn=0 時無影響
加入惰性氣體（恆容）無影響❌ 不變各組分分壓不變
加入惰性氣體（恆壓）向氣態 mol 數較多的一端❌ 不變相當於減壓
升高溫度向吸熱方向（ΔH > 0 那端）✅ 改變！（吸熱 → K 增大）唯一可改變 Keq 的操作
加入催化劑無影響❌ 不變只加快達到平衡的速率
移除固態（如 CaCO₃）無影響❌ 不變固態不計入 Keq
💡 熱力學與動力學產物的區別
動力學產物（Kinetic product）：活化能低，反應速率快，低溫時優勢，可能是少取代烯/烯醇
熱力學產物（Thermodynamic product）：熵/焓上更穩定，高溫或長時間反應優勢，通常是多取代烯/更穩定結構
例：E2 低溫→動力學（Hofmann）；高溫平衡→熱力學（Zaitsev）

6.1 酸鹼計算系統 ACID-BASE CALCULATIONS

── 基本關係 ──
pH = −log[H⁺]；pOH = −log[OH⁻]；pH + pOH = 14（25°C）
Kw = [H⁺][OH⁻] = 1.0×10⁻¹⁴（25°C）
Ka × Kb = Kw（共軛酸鹼對）；pKa + pKb = 14

── Henderson-Hasselbalch（緩衝必用）──
pH = pKa + log([A⁻]/[HA])
半當量點：[A⁻] = [HA] → pH = pKa（滴定曲線中間）
有效緩衝範圍：pH = pKa ± 1（即比例介於 0.1 ~ 10）
溶液類型計算公式注意事項
強酸（如 HCl）pH = −log C（完全解離）C < 10⁻⁶ M 時需考慮水的自電離
弱酸 HA$[H^+] \approx \sqrt{K_a \cdot C}$
（x ≪ C 時近似）		驗證：x/C < 5% 才可用近似法
弱酸鹽（如 CH₃COONa）$[OH^-] = \sqrt{K_b \cdot C} = \sqrt{\frac{K_w}{K_a} \cdot C}$水解，pH > 7
緩衝溶液Henderson-Hasselbalch加入少量強酸/強鹼，pH 變化小
多質子酸（H₂A）第一步 Ka₁ ≫ Ka₂ → 只計第一步等電點 pI = (pKa₁ + pKa₂)/2
⚠ 極稀強酸 pH 計算陷阱（考試常考！）
題目：5×10⁻⁸ M HCl 的 pH = ?
錯誤解法：pH = −log(5×10⁻⁸) ≈ 7.3 → 錯！（酸溶液 pH 不能 > 7）
正確解法：需聯立水的自電離：[H⁺] = [Cl⁻] + [OH⁻]
解方程：[H⁺]² − 5×10⁻⁸[H⁺] − Kw = 0 → [H⁺] ≈ 1.03×10⁻⁷ M → pH ≈ 6.97（仍為酸性）

6.2 有機化合物酸鹼強度 ORGANIC ACID-BASE STRENGTH

有機酸強度順序（pKa 由小到大，即酸性由強到弱）

強酸性（pKa ~1–5）：磺酸 RSO₃H；羧酸 RCOOH（4–5）

中酸性（pKa ~10）：酚 ArOH

弱酸性（pKa ~15–16）：水（15.7）；醇 ROH（~16）

極弱酸（pKa ~25）：末端炔 ≡C-H；酮的 α-H（~20）

影響因素：共振穩定負離子（↑酸性）；吸電子基（↑酸性）；雜化（sp > sp² > sp³ 酸性）

有機胺鹼性（水溶液）— 常考陷阱

水溶液鹼性順序：脂肪 2° 胺 > 脂肪 1° 胺 > 脂肪 3° 胺 > NH₃ > 芳香胺 > 醯胺

⚠ 三甲胺的鹼性反而比二甲胺弱！
氣相：(CH₃)₃N > (CH₃)₂NH（電子效應）
水溶液：(CH₃)₂NH > (CH₃)₃N（三甲胺空間阻礙大，N 難被溶劑化，pKb 較大）
考試通常問水溶液鹼性，切勿混淆氣相和水溶液結果！

6.3 電化學 ELECTROCHEMISTRY

── 電池電位計算 ──
E°cell = E°(cathode, 還原電位) − E°(anode, 還原電位)
ΔG° = −nFE°cell （F = 96485 C/mol；n = 轉移電子數）
E°cell > 0 → ΔG° < 0 → 反應自發

── Nernst 方程（非標準條件）──
E = E° − (RT/nF) ln Q = E° − (0.0592/n) log Q （25°C）
平衡時 E = 0：E° = (0.0592/n) log K

── Faraday 定律 ──
mol e⁻ = Q/F = It/F ； mol substance = mol e⁻ / n
電極原電池（Galvanic cell）電解槽（Electrolytic cell）
陽極 Anode氧化（Oxidation）；負極（電子流出）氧化（Oxidation）；接外部電源正極
陰極 Cathode還原（Reduction）；正極（電子流入）還原（Reduction）；接外部電源負極
ΔGΔG < 0（自發），E° > 0ΔG > 0（非自發），需外加電能
代表例Zn-Cu Daniell cell；鋰電池（放電）電解食鹽水；電鍍
💡 AN OX RED CAT — 永遠記住！
ANode = OXidation（陽極發生氧化）
REDuction at CAThode（陰極發生還原）
原電池與電解槽均適用此記憶法，差別只在陽極接負還是正電源。

7.1 速率定律與反應級數 RATE LAW & REACTION ORDER

⚠ 最重要原則：速率定律必須由實驗決定！
對於反應 aA + bB → C：速率 = k[A]ˣ[B]ʸ 中的 x, y 不一定等於 a, b。
只能透過實驗（不同初始濃度測速率）確定 x 和 y，絕不能從化學計量係數直接推導。
級數速率定律積分式半衰期 t₁/₂線性圖
零級rate = k[A] = [A]₀ − ktt₁/₂ = [A]₀/(2k)（與濃度有關）[A] vs t（斜率 = −k）
一級rate = k[A]ln[A] = ln[A]₀ − ktt₁/₂ = ln2/k（與濃度無關！）ln[A] vs t（斜率 = −k）
二級rate = k[A]²1/[A] = 1/[A]₀ + ktt₁/₂ = 1/(k[A]₀)（與濃度成反比）1/[A] vs t（斜率 = k）
Arrhenius 方程與催化劑的本質
k = A · e^(−Ea/RT) （A = 指前因子；Ea = 活化能）
ln(k₂/k₁) = (Ea/R)(1/T₁ − 1/T₂) （比較兩溫度的 k）
催化劑的作用：降低活化能（Ea），使反應速率增大
催化劑不改變：ΔH、ΔG°、Keq、反應物/產物結構
正反應與逆反應速率同比例增加 → 平衡位置不變
💡 一級反應的實際應用
放射性衰變（I-131 t₁/₂ = 8天；C-14 定年法）
某些藥物代謝（零級/一級切換）
SN1 反應速率（一級：Rate = k[RX]）
n 個半衰期後剩餘量 = (1/2)ⁿ × 初始量

7.2 量子數與電子組態 QUANTUM NUMBERS & ELECTRON CONFIGURATION

量子數符號代表意義允許值考試重點
主量子數n電子殼層（大小/能量）1, 2, 3, 4…n 越大，能量越高
角量子數l軌域形狀（亞層）0(s), 1(p), 2(d), 3(f)；最大 n−1各亞層軌域數 = 2l+1
磁量子數ml軌域在空間的方向−l, …, 0, …, +lp 有 3 個；d 有 5 個
自旋量子數ms電子自旋+½ 或 −½Pauli：每軌域最多 2 個（自旋相反）
填充規則（三大定律）
  • Aufbau 原理：能量低的軌域優先填（1s→2s→2p→3s→3p→4s→3d→4p…）
  • Pauli 不相容：每個軌域最多 2 個電子，且自旋相反
  • Hund 規則：同能量軌域先各填一個電子（自旋同向），再配對
過渡金屬特殊電子組態
  • Cr：[Ar]3d⁵4s¹（非 3d⁴4s²）→ 3d 半充滿更穩定
  • Cu：[Ar]3d¹⁰4s¹（非 3d⁹4s²）→ 3d 全充滿更穩定
  • Mo（d⁵）、Ag（d¹⁰）同樣例外
  • MO 理論鍵級 = (成鍵e⁻ − 反鍵e⁻)/2

7.3 週期律趨勢 PERIODIC TRENDS

性質同週期（左→右）同族（上→下）重要例外
原子半徑↓ 減小（有效核電荷增加）↑ 增大（電子殼層增加）過渡金屬變化緩慢
游離能↑ 增大（一般趨勢）↓ 減小Be > B；N > O（詳見下框）
電子親和力↑ 增大（傾向）↓ 減小F < Cl（F 原子小，電子排斥）
電負度↑ 增大↓ 減小F 最大（3.98）
⚠ 游離能例外——每年必考！
Be > B：Be 的 2s 軌域電子比 B 的 2p 電子難移除（2s 能量較低，穿透性強）
N > O：N 的 2p 三個軌域各有一個電子（半充滿），特別穩定；
O 的一個 2p 軌域有兩個電子（配對），排斥↑，反而更易移除一個電子

同理第三週期：Mg > Al（3s vs 3p）；P > S（3p 半充滿）

第二游離能跳升：若第一游離能移走了成對電子，第二游離能會突然大幅增加（例：Na 的第二游離能遠大於第一）

8.1 ¹H NMR 系統解析法 ¹H NMR ANALYSIS

NMR 四步驟解題法（按此順序，不跳步）
  1. 積分（Integration）：各訊號面積比 → 各類質子的相對數量。若已知分子式，可換算絕對數量。
  2. 化學位移（δ, ppm）：確認每組質子所處化學環境（判斷官能基）。
  3. 裂分模式（Splitting Pattern）：n+1 規則 → 推算相鄰碳上的 H 數目 → 推測相鄰碳結構。
  4. 偶合常數（J value）：trans-烯 J ≈ 12–18 Hz；cis-烯 J ≈ 6–12 Hz → 判斷幾何。
¹H NMR 化學位移範圍 (δ, ppm) — 可視化速查
⚠ n+1 裂分規則的常見錯誤
正確：相鄰 n 個不等價 H → 裂分為 n+1 重峰（singlet, doublet, triplet, quartet…）
錯誤思維：「相鄰幾個 H 就裂分幾峰」（忘記 +1）

例：-CH₂-CH₃ 的 CH₃：相鄰 CH₂ 有 2 個 H → 裂分為 3 重峰（triplet）
例：-CH₂-CH₃ 的 CH₂：相鄰 CH₃ 有 3 個 H → 裂分為 4 重峰（quartet）

等價質子不互相裂分！如 CH₃CH₃（乙烷）的所有 H 等價 → 只有 singlet

8.3 質譜 (MS) 與 UV-Vis 概念 MS & UV-Vis

質譜 Mass Spectrometry
  • 分子離子峰 M⁺ → 分子量（整數）
  • M+1 峰 → ¹³C（M的1.1%×碳數）；¹⁵N, ²H
  • M+2 峰（~1:1）→ 含溴（⁷⁹Br/⁸¹Br）
  • M+2 峰（~3:1）→ 含氯（³⁵Cl/³⁷Cl）
  • 碎片峰 → 分子特定鍵斷裂，推測結構
  • Base peak（基礎峰）= 最強峰（100%）
UV-Vis 紫外可見光譜
  • π→π* 躍遷（共軛系統，200–400 nm）
  • n→π* 躍遷（羰基孤對 e⁻，~270 nm，弱）
  • 共軛延長 → λmax 紅移（bathochromic shift）
  • Beer-Lambert：A = εlc（ε=莫耳消光係數）
  • 極性溶劑：n→π* 藍移；π→π* 紅移
  • λmax 可推估共軛程度
練習題 — 綜合光譜推結構★★★ 常考型

分子式 C₄H₈O，光譜數據如下：

  • IR：~1715 cm⁻¹ 強吸收；無 O-H 寬峰；無醛基特徵雙峰（~2720, 2820）
  • ¹H NMR：δ 2.4（s, 3H）、δ 2.6（q, 2H）、δ 1.1（t, 3H）
  • 分子式不飽和度 = 1

此化合物最可能為何？

✅ 答案：methyl ethyl ketone（MEK，2-丁酮，CH₃COCH₂CH₃）
解析：
• IR 1715 cm⁻¹ → 酮 C=O；無 O-H → 非醇；無醛特徵 → 非醛
• δ 2.4 (s, 3H) → CH₃CO-（甲酮甲基，無鄰 H，故為 singlet）
• δ 2.6 (q, 2H) → -CH₂-（被相鄰 CH₃ 3個H 裂分 → quartet）
• δ 1.1 (t, 3H) → -CH₃（被相鄰 CH₂ 2個H 裂分 → triplet）
結構：CH₃-CO-CH₂-CH₃ ✓（分子式 C₄H₈O，不飽和度 1 ✓）

A-1 · 必背公式速查表
── 熱力學 Thermodynamics ──
ΔG = ΔH − TΔS
ΔG° = −RT ln Keq
ΔG° = −nFE°
ΔG = ΔG° + RT ln Q
Hess: ΔH = Σ ΔHf(P) − Σ ΔHf(R)
Kp = Kc(RT)^Δn
── 電化學 Electrochemistry ──
E°cell = E°(cathode) − E°(anode)
E = E° − (0.0592/n)logQ (25°C)
ΔG° = −nFE°cell
mol e⁻ = It/F ; F ≈ 96500 C/mol
ΔG° = −RT ln K
── 動力學 Kinetics ──
一級：ln[A] = ln[A]₀ − kt
一級 t₁/₂ = ln2/k = 0.693/k（與濃度無關）
k = Ae^(−Ea/RT)（Arrhenius）
ln(k₂/k₁) = (Ea/R)(1/T₁ − 1/T₂)
MO 鍵級 = (成鍵e⁻ − 反鍵e⁻)/2
── 酸鹼 / 溶液 Acid-Base ──
pH = pKa + log([A⁻]/[HA])
Kw = Ka × Kb = 10⁻¹⁴ (25°C)
pI = (pKa₁ + pKa₂)/2（多質子酸等電點）
ΔTb = iKbm；ΔTf = iKfm
π = iMRT（滲透壓）
