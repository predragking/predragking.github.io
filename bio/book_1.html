7-1 Hardy-Weinberg 平衡計算
H-W 公式（必背）
p + q = 1　（等位基因頻率之和）
p² + 2pq + q² = 1　（基因型頻率之和）
遷移後：q' = (1−m)·q_局 + m·q_移
例：m=0.1, q_局=0.8, q_移=0.2
→ q' = 0.9×0.8 + 0.1×0.2 = 0.72+0.02 = 0.74
⚠️ 不可直接取平均(0.8+0.2)/2=0.5！須用加權平均
H-W 計算範例
苦味者4%為tt（隱性純合）→ q²=0.04
→ q=0.2, p=0.8
攜帶者Tt頻率 = 2pq = 2×0.8×0.2 = 0.32
TT頻率 = p² = 0.64（非0.2！）

❌ 肌肉分解肝糖後可補充血糖
✅ 肌肉缺乏Glucose-6-phosphatase，無法將G-6-P轉為游離葡萄糖釋出；不能補充血糖（只有肝腎可以）
T-31MalonylCoA 抑制（而非刺激）CPT（中國醫106）
❌ MalonylCoA 刺激 CPT（肉鹼醯基轉移酶）
✅ MalonylCoA 濃度↑ → 抑制 CPT-1 → 脂肪酸無法進入粒線體氧化（脂肪合成時同時抑制氧化）
T-32TCA 最多 CO₂ 釋放（義守114）
❌ CO₂ 最多在糖解或電子傳遞鏈釋放
✅ 大部分CO₂在TCA循環（每葡萄糖4 CO₂）釋放；電子傳遞鏈不產CO₂（產H₂O）
T-33Rotenone 抑制 Complex I（全校必考）
❌ Rotenone 抑制 Complex III 或 II
✅ Rotenone → 只抑制Complex I；Malonate→CII；Antimycin→CIII；CN⁻/CO→CIV；Oligomycin→CV
T-34Km 的意義（高頻）
❌ Km↑ 代表親和力增加
✅ Km↑ = 親和力↓（需更高底物濃度達½Vmax）；kcat/Km = 催化效率常數
T-35Leu/Lys 不可糖質新生（清華/中興）
❌ 所有胺基酸都可用於糖質新生
✅ Leu 和 Lys 是純生酮胺基酸，不能轉為葡萄糖；可用於糖質新生的：Ala, Gly, Ser, Gln, Asp, 丙酮酸, 草醋酸, 甘油等
T-36棕色 vs 白色脂肪（亞洲大學）
❌ 棕色和白色脂肪都能解偶聯產熱
✅ 棕色脂肪含UCP-1（解偶聯蛋白）→ 解偶聯→產熱（非ATP合成）；白色脂肪主要儲能，無此功能
T-37邦克列酸（Bongkrekic acid）毒素機制（中國醫114）
❌ 邦克列酸攻擊呼吸系統造成窒息
✅ 邦克列酸抑制粒線體的ADP/ATP 轉運蛋白（ANT）→ ATP 無法從粒線體釋出 → 細胞能量代謝崩潰
T-38NO 的訊號路徑（中國醫106）
❌ NO 刺激 PDE（磷酸二酯酶）使平滑肌放鬆
✅ NO → 活化 soluble guanylate cyclase → cGMP↑ → PKG → 肌肉放鬆；PDE 是分解 cGMP 的（Sildenafil抑制PDE）
T-39Tay-Sachs 遺傳方式（高醫/中興）
❌ Tay-Sachs 是顯性遺傳；發生在線粒體
✅ Tay-Sachs 是隱性遺傳；Hexosaminidase A 缺乏導致 Ganglioside 堆積在溶酶體（非線粒體）
T-40GAGs 的電荷（高醫/中興）
❌ 糖胺聚糖（GAGs）帶正電荷
✅ GAGs 含硫酸根或羧基，帶負電荷；因此能吸引大量Na⁺和水分，使ECM具有抗壓性

✅ CAM 夜間用PEP carboxylase固定CO₂（白天才由Rubisco利用）；C4是「空間分隔」；CAM是「時間分隔」

❌ 長期開燈→縮短光期→短日照植物提前開花
✅ 長期開燈→縮短暗期→提前開花→是長日照植物（需短暗期才開花）

❌ 細胞質流靠微管（microtubule）驅動

❌ 夜間根部中柱水勢低于皮層→水從葉往根流

T-47子房 vs 胚珠（高醫112）
❌ Ovaries→種子；Ovules→果實
✅ Ovules（胚珠）→ 種子；Ovaries（子房）→ 果實（順序絕對不能搞反）
T-48粒線體內共生起源（中國醫114）
❌ 粒線體來自革蘭氏陽性菌
✅ 粒線體起源於革蘭氏陰性菌（α-變形菌門，如立克次氏體）；雙層膜結構與革蘭氏陰性菌外膜相符

✅ Pasteur Effect = 有氧環境抑制糖解速率，丙酮酸優先進TCA循環；低pH使酵素活性↓→乳酸堆積

T-55MRSA 抗藥機制（中國醫114）

❌ 分子時鐘可使用旁系同源基因（paralogs）
✅ 必須用直系同源基因（orthologs）；paralogs 是基因複製產生，不反映物種間演化距離

✅ 天擇作用於個體的表現型（phenotype），通過改變個體存活率/繁殖率間接改變族群等位基因頻率

Hardy-Weinberg 族群遺傳
p + q = 1；p² + 2pq + q² = 1
攜帶者頻率 = 2pq
q² = 病例頻率 → q = √q² → p = 1−q
遷移後：q′ = (1−m)·q_局 + m·q_移
適合度加權：w̄ = p²w_AA + 2pq·w_Aa + q²w_aa
p′ = [p²w_AA + pq·w_Aa] / w̄
Michaelis-Menten 酵素動力學
v = Vmax·[S] / (Km + [S])
Lineweaver-Burk：1/v = (Km/Vmax)·(1/[S]) + 1/Vmax
y截距 = 1/Vmax；x截距 = −1/Km
斜率 = Km/Vmax
例：Vmax=30, Km=50, [S]=25 → v = 10 µM/min
脂肪酸 β-氧化計算
C16:0（棕櫚酸）：7輪 → 8 Acetyl-CoA + 7 FADH₂ + 7 NADH
C18:0（硬脂酸）：8輪 → 9 Acetyl-CoA + 8 FADH₂ + 8 NADH
Cn:0 通式：(n/2−1) 輪 → n/2 Acetyl-CoA
活化耗：−2 ATP（→ AMP + PPi）
心臟輸出量與血壓
CO = SV × HR
CO = (EDV − ESV) × HR
EF（射血分率）= (EDV − ESV) / EDV
MAP = CO × TPR
F = ΔP / R；R ∝ 1/r⁴
生態能量（GPP/NPP）
GPP = NEE + Ra + Rh
NPP = GPP − Ra
例：NEE=90, Ra=40, Rh=20 → GPP = 150
族群幾何成長
N_t = N_0 × λ^t
例：N_0=1000, λ=3, t=3 → N_3 = 27,000
χ² = Σ[(O−E)²/E]
df = 類別數 − 1；α=0.05, df=1 → 臨界值3.84
重組率計算
重組率(%) = 重組型後代數 / 總後代數 × 100
1 cM = 1% 重組頻率
⚠️ 斥型：重組外表型 = AB + ab（各5%合計10%，非20%）
